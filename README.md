#### Запуск приложения:

```
Клонируем проект:
git clone https://github.com/AnastasiyaMAV/ScheduleForTheWeek.git

Переходим в проект:
cd ScheduleForTheWeek

Устанавливаем зависмости:
npm install

Запускаем проект:
npm run dev

Открываем http://localhost:5173/ScheduleForTheWeek
```
